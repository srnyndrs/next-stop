# Next Stop
An Android application providing real-time public transport informations in Budapest, including bus, tram, metro, and other transit services.

## Technologies
- Jetpack Compose
- Ktor
- Room Database
- DataStore
- Dagger Hilt
- Google Maps
- Fused Location Provider
- Accompanist Permissions
- Clean Architecture
